# App Structure

- `routes/web.php`: public, auth, app, and admin route groups.
- `app/Http/Controllers/Auth`: signup, sign in, password help, first admin setup.
- `app/Http/Controllers/App`: Interest, posts, profile, follows, privacy, modules.
- `app/Http/Controllers/Admin`: Admin Zone dashboard and sections.
- `database/migrations`: users plus Sirraty core platform tables.
- `resources/views`: public, auth, app, and admin Blade views.
